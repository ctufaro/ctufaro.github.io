Click the Spread Gossip button below to simulate an initial number of nodes spreading a message to a random number of receiving nodes. Click the Reload button to reshuffle the network and to re-apply the configuration settings below.

Nodes represent the total number of nodes on the network. The simulator defaults to 100. Fanout represents the number of nodes a single gossiping node will reach out to. Init represents the initial number of gossiping nodes when the network starts. Edges, a read-only value that represents a single link between two gossiping nodes.
